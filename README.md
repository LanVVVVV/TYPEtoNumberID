# TYPEtoNumberID

TYPEtoNumberID is a mod for **Monster Black Market** —— aimed at expanding the supported number of Spine attachment types for the information panel.

### Features

* Change the type identifiers in the information panel from letters to numbers.
* Automatically extract all attachment types and switching methods for the Gallery.

### Requirements

| Dependency           | Version |
| -------------------- | ------- |
| MBM.ModLoader        | latest  |
| Monster Black Market | latest  |

### Compatibility

- ✅ **TitsMod** — prevent duplicate Tits switching in gallery.
